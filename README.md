## Project Summary
This project presents the design, simulation, and manufacturing of a 3D Printed Manual Shearing Machine developed using CAD modelling and additive manufacturing techniques. The machine is designed for lightweight sheet and paper cutting applications while demonstrating mechanical design principles, lever mechanism operation, and structural analysis.

## Project Overview

The objective of this project was to design and manufacture a lightweight manual shearing machine using 3D printing technology. The machine operates using a lever mechanism to amplify manual force for cutting thin materials.

The project demonstrates the practical implementation of:
- Mechanical Design
- CAD Modeling
- Assembly Design
- Structural Simulation
- Additive Manufacturing

## Project Development Process

### CAD Modeling
The complete shearing machine components were designed using SolidWorks CAD software. Parametric modeling techniques were used for accurate part creation and modification.

### Assembly
All individual components were assembled using SolidWorks assembly mates and alignment constraints. The assembly was verified for proper movement and functionality.

### Simulation Analysis
Structural simulation was performed using SolidWorks Simulation to analyze stress, strain, and displacement under an applied load of 88.3 N.

### 3D Printing Process
The CAD models were converted into STL format and prepared using FlashPrint software for additive manufacturing and prototype development.
